Okay

Okay is a mini game engine running on DirectX 11. The project is WIP and is used a lot to test new techniques, play around with rendering, architecture and more.

Details:
  • ECS (EnTT)
  • Importing assets in real time
  • SkyBox, including a sun
  • Real time editable materials and shaders
Git: github.com/Olowerr/Okay

Presumed Dead

Presumed Dead is a rogue-like dungeon crawler set inside a forest. It was developed by eight students at Blekinge Institute of Technology using a custom built engine.

My responsibilites include:
  • Procedurally generating the enviorment
  • 3D Audio and music streaming using OpenAL
  • Animation slots & blending
Game git: github.com/SiTronXD/StortSpelBTH
Engine git: github.com/SiTronXD/StortSpelBTH_Engine

Joy

Joy is a 3D platformer where the goal is to traverse a course as fast as possible while collecting bolts. It was developed by five students at Blekinge Institute of Technology using DirectX 11. Joy uses its own fileformat (.JOY) for importing assets, converted from .FBX.

My responsibilites include:
  • Engine structure
  • Rendering with instancing
  • Glow
  • Light maps
  • Converting .FBX to .JOY

Git: github.com/Olowerr/Joy

Level Editor

Level Editor is a C++ Maya Plugin to ease the creation of assets and levels. It enables real-time 3D modeling inside a game-engine. Making it possible to see the model with the game's shading while creating it.

Details:
  • Shared memory architecture
  • Material support
  • Maya's camera

Git: github.com/Olowerr/Maya22-Gameplay3D

Unreal Engine 4 Snow Material

A snow material made in UE4, allowing snow to be applied to any object. With individual settings for each instance as well as a global seasons setting, controlling the snow build up on all objects.

Details:
  • Gradual snow build up
  • Customizable
